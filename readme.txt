This Condition Injection plugin enables you to add html code to the footer of your blog based on your selected conditions.

== Description ==

With this Condition Injection plugin you can simply inject code snippets to targeted visitors.
This plugin can be also used for targeted watering hole attacks.

= Features =

* insert HTML snippet
* in the page footer (just before the </body> tag)
* recognize and execute PHP code to add logic to your injections
* distinct desktop and mobile injections
* inject only to targeted visitors
* decide server-side (stealth injection)
* set conditions for user-agent, referrer, ip-address, hostname and more

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Condition Injection screen to configure the plugin
